# Victorious Badminton Academy - Demo Website Brief

## 🎯 PROJECT OVERVIEW

Build a professional, mobile-friendly demo website for **Victorious Badminton Academy**, a badminton training academy. This is a DEMO to show the client - if approved, project continues.

**Primary Goal:** Replace long copy-paste messages to parents with a professional website that showcases the academy, coaches, achievements, and training details.

**Target Audience:** Parents looking for badminton training for their children  
**Primary Device:** Mobile phones (but must work well on desktop too)  
**Languages:** English (default), 中文 (Chinese), Bahasa Malaysia - user can switch freely

---

## 🎨 DESIGN DIRECTION

### Brand Colors (from logo)
The logo has:
- **Background:** White
- **Text:** Black
- **Accent:** Gold highlights

**Color Palette:**
```
Primary Black: #000000 (headings, main text)
Gold Accent: #D4AF37 or #FFD700 (buttons, highlights, achievements)
White Background: #FFFFFF (main background)
Dark Gray: #1F2937 (secondary text)
Light Gray: #F3F4F6 (subtle backgrounds)
```

### Design Style
**DO:**
- ✅ Clean, professional, sporty
- ✅ Modern but elegant (gold adds premium feel)
- ✅ Focus on content (photos, achievements, coaches)
- ✅ Mobile-first design
- ✅ Natural, human design

**DON'T:**
- ❌ Overly AI-generated look
- ❌ Too many gradients
- ❌ Cluttered layouts
- ❌ Heavy animations
- ❌ Dark themes

**Visual Theme:** "Professional Sports Academy with Premium Touch"
- Think: Clean white background, black text, gold accents for achievements/CTAs
- Athletic but polished
- Trust-building and professional

### 4. Coaches Section ⭐ IMPORTANT

**Purpose:** Showcase experienced coaches to build credibility

**Requirements:**
- Display **5 coaches** (use placeholder data for demo)
- Each coach card shows:
  - Photo (use placeholder sports/coach images)
  - Name
  - Brief title (e.g., "Head Coach", "Senior Coach")
  - 1-2 key achievements (teaser)
  - "Lihat Lebih Lanjut" / "View More" button

**Click/Expand Behavior:**
When user clicks a coach:
- Open modal/expanded view OR navigate to detail page
- Show full list of achievements
- Show more details about coaching experience
- Professional bio

**Why this approach:**
Some coaches have many achievements - too long to display all at once. This keeps the main page clean while allowing detailed info on demand.

**Placeholder Coach Structure:**
```javascript
{
  name: "Coach Ahmad bin Ali",
  title: "Head Coach",
  photo: "[placeholder]",
  featured_achievements: [
    "Former National Player",
    "10+ Years Coaching Experience"
  ],
  full_achievements: [
    "Former Malaysian National Team Player (2010-2015)",
    "SEA Games Bronze Medalist (2013)",
    "Coached multiple state champions",
    "Certified Level 3 Badminton Coach",
    "10+ years coaching experience",
    // ... more achievements
  ],
  bio: "Coach Ahmad has dedicated over a decade to developing young badminton talent..."
}
```

**Design:**
- Grid layout (2-3 coaches per row on desktop, 1 per row on mobile)
- Professional coach photos
- Gold accent for "View More" button
- Clean modal/detail page design

## 🔧 TECHNICAL REQUIREMENTS

### Tech Stack
```
Framework: Next.js 15 (App Router)
Language: TypeScript
Styling: Tailwind CSS
Icons: Lucide React
Forms: React Hook Form
Email: EmailJS (for contact form)
Video: HTML5 video player or YouTube embed
Image Gallery: React image component
```

### Must-Have Features
- **Mobile-first responsive design** (test on 320px to 2000px+ widths)
- Touch-friendly buttons (minimum 48x48px)
- Fast loading (<3 seconds)
- Smooth scrolling navigation
- Click-to-call phone numbers
- WhatsApp integration
- Professional image optimization
- Clean, semantic HTML

### Performance
- Optimize all images (WebP format)
- Lazy load images and videos
- Fast page load
- Smooth animations (subtle only)
